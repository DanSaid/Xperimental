INFORMACIÓN

DESCRIPCIÓN: El curso se enfoca en desarrollar las habilidades necesarias para el dibujo y producción de planimetría utilizando el software de AutoCAD y todas las herramientas indispensables para aplicar diferentes técnicas de expresión y así poder crear de manera adecuada láminas para la presentación de proyectos arquitectónicos. Además se mostrará la manera correcta de administrar el flujo de trabajo para optimizar el tiempo de ejecución.
